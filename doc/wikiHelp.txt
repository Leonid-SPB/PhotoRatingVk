 
==Описание==
Приложение формирует рейтинг лучших фотографий пользователя или группы на основе количества лайков.  В рейтинге участвуют фотографии, которые понравились хотя бы одному пользователю. Для отображения отбирается не более 500 фотографий&#33; 

==Как пользоваться==

1. '''Выберите для кого составлять рейтинг''': 
* друг из списка "Пользователь"; 
* группа из списка "Группа"; 
* или введите в поле "ID" идентификатор пользователя или группы. Например: 'tproger' (группа Типичный Программист), 'club32411977' (группа Помощь в API VK), 'durov' (Павел Дуров) или 'id105876' (Леонид Азаренков). 

2. '''Нажмите кнопку "Рейтинг&#33;"'''. Начнется процесс формирования рейтинга, ход процесса отображается на шкале состояния рядом с кнопкой. 

3. '''Как только все данные будут загружены, на экране начнут появляться миниатюры отобранных фотографий в порядке убывания рейтинга.''' 

4. '''Любую фотографию можно''': 
* увеличить щелчком мыши по иконке лупы в правом верхнем углу; 
* открыть в отдельном окне щелчком мыши по фотографии. 

5. '''Делитесь лучшими фото с друзьями кнопкой 'Сохранить на стену'.''' Перейдя по ссылке из записи на вашей стене, друзья смогут увидеть ваши лучшие фотографии (или лучшие фотографии выбранной группы/пользователя). 

'''В случае возникновения ошибок''': 
* проверьте, что блокировщики рекламы/скриптов настроены правильно и не мешают работе приложения (не блокирует всплывающие окна и скрипты); 
* попробуйте очистить кэш браузера и перезагрузить приложение (CTRL+F5).
